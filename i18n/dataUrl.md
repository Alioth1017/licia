## CN

解析序列化 Data URL。

### parse

解析 Data URL。

|参数名|说明|
|-----|---|
|dataUrl|Data URL 字符串|
|返回值|解析后的对象|

### stringify

将数据转化为 Data URL。

|参数名|说明|
|-----|---|
|data|要转化的数据|
|mime|mime 类型|
|options|Stringify options|
|返回值|Data url string  |

### 选项

|选项名|说明|
|-----|---|
|base64=true|是否是 base64 编码|
|charset|字符集|
